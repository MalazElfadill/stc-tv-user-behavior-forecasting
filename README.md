# STC TV User Behavior Forecasting

## Project Overview
This project uses historical STC TV watch-time data to build a simple forecasting model that predicts customer watch behavior over the next 60 days.

## Objective
- Forecast future watch hours
- Identify peak usage periods
- Support decision-making with data insights

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Methodology
1. Data cleaning
2. Date transformation
3. Feature engineering
4. Linear Regression forecasting
5. Visualization
6. Peak demand analysis

## Results
The model suggested a downward trend in watch hours over time. Peak watch-time dates were identified to support promotions, content planning, and operational readiness.
